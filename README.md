# ReCharge - Smart PC Monitoring & Energy Management

ReCharge คือระบบบริหารจัดการพลังงานและตรวจสอบสถานะคอมพิวเตอร์ระยะไกล (Remote PC Monitoring) แบบครบวงจร พัฒนาขึ้นเพื่อช่วยให้ผู้ใช้สามารถติดตามการทำงานของฮาร์ดแวร์ ควบคุมเครื่องระยะไกล และคำนวณค่าไฟฟ้าที่เกิดขึ้นจากการใช้งานคอมพิวเตอร์ได้อย่างแม่นยำ ผ่าน Web Application ที่เชื่อมต่อกับระบบ Cloud แบบ Real-time

ทดลองใช้งาน Web App: [https://lololo256.github.io/recharge](https://recharge-psi-sandy.vercel.app/)

---

## สถาปัตยกรรมระบบ (System Architecture)
โปรเจกต์นี้ทำงานร่วมกัน 3 ส่วนหลัก:
* **Client Agent (Python):** โปรแกรม recharge_sender.exe ฝังตัวทำงานเบื้องหลัง (Background Service) บน Windows คอยดึงข้อมูลฮาร์ดแวร์ (CPU, GPU, RAM, Temp) และส่งขึ้น Cloud
* **Cloud Database (Firebase Realtime Database):** ทำหน้าที่เป็นตัวกลางรับส่งข้อมูลระหว่างคอมพิวเตอร์และ Web Application ด้วยความหน่วงต่ำ (Low Latency)
* **Web Application (HTML/JS/CSS):** แดชบอร์ดแสดงผลกราฟิก (Chart.js) และเป็นแผงควบคุม (Control Panel) ให้ผู้ใช้สั่งการจากมือถือหรืออุปกรณ์ใดก็ได้

---

## ฟีเจอร์หลัก (Core Features)
* **Real-time Hardware Monitoring:** ตรวจสอบการใช้งาน CPU, GPU, RAM, พื้นที่ฮาร์ดดิสก์, ความเร็วอินเทอร์เน็ต และอุณหภูมิเครื่องแบบวินาทีต่อวินาที
* **Energy & Cost Estimation:** ระบบคำนวณค่าไฟอัจฉริยะ (อิงจาก TDP ของสเปคเครื่อง เช่น จำนวนพัดลม, ประเภท HDD/SSD) พร้อมสรุปยอดรายวัน รายเดือน และรายปี
* **Smart Gaming Mode:** ตรวจจับการเล่นเกมอัตโนมัติ (เช่น Valorant, GTA V, Battlefield) เพื่อแยกคำนวณระยะเวลาและค่าไฟเฉพาะตอนเล่นเกม
* **Remote PC Control:** สั่ง ปิดเครื่อง (Shutdown), รีสตาร์ท (Restart), สลีป (Sleep) หรือ ล็อคหน้าจอ (Lock) จากระยะไกล
* **Remote Task Killer:** สแกนหาโปรแกรมที่กิน CPU สูงสุด และสามารถกด "Force Close" ปิดโปรแกรมนั้นจากมือถือได้ทันที
* **Sleep Timer:** ตั้งเวลาปิดเครื่องอัตโนมัติ พร้อมแสดงเวลานับถอยหลังบนหน้าเว็บ

## ปัญหาที่พบระหว่างพัฒนา และแนวทางการแก้ไข (Engineering Problem Solving)
ในการพัฒนาระบบที่ต้องเชื่อมต่อระหว่าง Hardware, OS และ Cloud พบปัญหาเชิงวิศวกรรมหลายประการ ซึ่งได้แก้ไขตามหลักการทางวิทยาการคอมพิวเตอร์ ดังนี้:

### 1. ปัญหา Race Condition (ข้อมูลชนกัน)
* **อาการ:** ตัวแปรตรวจจับเมาส์ถูกเขียนและอ่านพร้อมกันจาก 2 Thread ทำให้โปรแกรมทำงานผิดพลาด
* **วิธีแก้:** ใช้ threading.Lock() เข้ามาช่วยจัดการ Critical Section ป้องกันไม่ให้ Thread แย่งกันใช้ทรัพยากร

### 2. ปัญหา Memory / Variable Overflow ในการคำนวณค่าเฉลี่ย
* **อาการ:** การหาค่าเฉลี่ยวัตต์โดยใช้ผลรวมสะสมหารด้วยจำนวนครั้ง เมื่อเปิดเครื่องนานเป็นสัปดาห์ ตัวเลขจะใหญ่จนผิดเพี้ยน
* **วิธีแก้:** เปลี่ยนไปใช้อัลกอริทึม Exponential Moving Average (EMA) เพื่อกรองข้อมูลให้สะท้อนค่าปัจจุบันเสมอ และใช้หน่วยความจำคงที่

### 3. ปัญหาสิทธิ์และหน้าจอกระพริบ (CMD Popup)
* **อาการ:** การดึงข้อมูลฮาร์ดแวร์บางส่วนต้องใช้คำสั่ง PowerShell ทำให้หน้าต่างสีดำกระพริบขึ้นมากวนใจผู้ใช้ตอนเล่นเกม
* **วิธีแก้:** เพิ่ม Parameter CREATE_NO_WINDOW เพื่อบังคับให้ Process ทำงานแบบ Silent mode

### 4. ปัญหา Fork Bomb (ระบบแยกร่างไม่รู้จบ) ตอนแพ็กเป็น .exe
* **อาการ:** การสั่งโปรแกรมรีสตาร์ทตัวเอง ทำให้เกิดลูปเปิดโปรแกรมซ้อนกันจน RAM เต็ม
* **วิธีแก้:** นำคำสั่งรีสตาร์ทออกจาก Global Scope และเพิ่มพารามิเตอร์ --hidden-import _overlapped ผ่าน PyInstaller เพื่อป้องกัน Library สูญหาย

### 5. ปัญหาระบบหลุดการเชื่อมต่อ (Firebase Listener Disconnect)
* **อาการ:** เมื่อคอมพิวเตอร์เข้าโหมด Sleep หรือเน็ตหลุด ตัวรับคำสั่งจากเว็บจะหยุดทำงาน
* **วิธีแก้:** เขียนระบบ Reconnect โดยใช้ try-except ครอบ Listener หากหลุดให้ล้างสถานะ Flag เพื่อให้ลูปหลักพยายามต่อใหม่โดยอัตโนมัติ

## วิธีติดตั้งและใช้งาน ReCharge Server (สำหรับ PC)
โปรแกรมนี้จะทำงานเบื้องหลังเงียบๆ เพื่อส่งข้อมูลสเปคคอมพิวเตอร์และค่าไฟไปแสดงผลที่หน้าเว็บของคุณ

### ขั้นตอนการติดตั้ง (ทำแค่ครั้งเดียวจบ)
1. **แตกไฟล์ (Extract):** นำโฟลเดอร์นี้ไปไว้ในที่ที่คุณต้องการ (แนะนำให้เก็บไว้ในไดร์ฟ D: หรือ Documents ไม่ควรวางไว้หน้า Desktop)
2. **เปิดโปรแกรม:** ดับเบิ้ลคลิกเปิดไฟล์ recharge_sender.exe (หาก Windows ขึ้นหน้าต่างสีฟ้าแจ้งเตือน ให้คลิกที่ "More info" แล้วกด "Run anyway")
3. **เชื่อมต่อระบบ:** จะมีหน้าต่างเด้งขึ้นมาให้กรอก "รหัสเครื่อง (UID)" ให้นำ UID จากหน้าเว็บของคุณมาใส่ แล้วกด OK
4. **เสร็จสิ้น:** หน้าต่างจะหายไป ระบบได้ทำการฝังตัวเองให้ทำงานอัตโนมัติเบื้องหลังเรียบร้อยแล้ว

### เกร็ดความรู้เพิ่มเติม
* โปรแกรมจะเริ่มทำงานเองอัตโนมัติทุกครั้งที่คุณเปิดคอมพิวเตอร์ (Auto-Start)
* หากต้องการเปลี่ยน UID ให้เปิดไฟล์ .exe นี้ซ้ำอีกครั้ง ระบบจะรีเซ็ตให้กรอกใหม่
* หากต้องการปิดโปรแกรมชั่วคราว ให้เปิด Task Manager (Ctrl+Shift+Esc) หาชื่อ recharge_sender.exe แล้วกด End Task
